Inspirations

For designing my nav bar, I looked at Jan's website as an example, and decided to make my links bigger once you click on it.

I also made it have an underline, as well as coloured so it isn't directly copying his style and shows I took inspiration from it.

For custom cursor, Jan also taught me how to implement custom cursor on certain areas of my website.

I reused some of my code from previous assignments, such as the navbar and table from my restaurant assignment,

the filter code on my images from flexbox gallery, and grid layout from my Canucks grid.


Problems I encountered...

I had to use some help with AI resources (ChatGPT) in order to make my website have proper margins, text displays,

and image sizes. I used AI to help solve my problems by prompting it with my previous HTML and CSS code and figuring

out how to correctly format my code so that everything shows up as I intended it to.


Things I made

I made it so my horizontal nav bar turns red and stays red once clicked. It becomes enlarged and underlined on the HTML

you are on. The horizontal navbar you used to get to this citation page is made to be sticky. This is when the bar stops going up

when you keep on scrolling, and sticks to the top of the screen for ease of access, no matter where you are on the website.

I put in a custom cursor that is Rocket League-themed, and made it change between a car and the logo once you hover

over all the links and the body, respectively.
